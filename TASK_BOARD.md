## 🧩 子任务格式

复杂任务可拆解为子任务，格式如下：

```
T030 (父任务)
├── T030-1 [P1] 前端布局修复 - ✅ 已完成
├── T030-2 [P1] 按钮事件修复 - 🟡 进行中
├── T030-3 [P2] 回归测试 - ⏳ 待执行
└── T030-4 [P3] 文档更新 - ⏳ 待执行
```

**子任务状态同步：**
- 子任务完成时自动更新父任务进度
- 所有子任务完成后，父任务自动标记为 REVIEW
