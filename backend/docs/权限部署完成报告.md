## 📊 权限配置 API

### 获取所有权限

```bash
curl -X GET "https://your-domain/permission/list" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 获取角色权限

```bash
curl -X GET "https://your-domain/permission/role/2" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 配置角色权限

```bash
curl -X PUT "https://your-domain/permission/role/2" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '[1,2,3,4,5,6]'
```

## ⚠️ 注意事项

1. **权限变更**：修改角色权限后，用户需重新登录生效
2. **系统管理员**：默认拥有所有权限，无需配置
3. **数据权限**：作业区数据权限通过 `user_work_areas` 表控制
4. **公开接口**：认证相关接口（`/auth/**`）已在拦截器中排除
