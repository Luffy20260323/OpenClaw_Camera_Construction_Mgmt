## 🔧 使用说明

### 系统管理员操作

1. **登录系统**
   - 用户名：`admin`
   - 密码：联系管理员获取

2. **配置角色权限**
   - 进入"系统管理" → "角色权限配置"
   - 选择角色
   - 勾选/取消权限
   - 填写备注
   - 保存

3. **查看审计日志**
   - 进入"系统管理" → "审计日志"
   - 使用筛选条件
   - 查看详情

### 开发人员使用

#### 后端添加权限控制

```java
@ApiPermission("user:create")
@PostMapping("/create")
public Result<User> createUser(...) {
    // ...
}
```

#### 前端添加权限控制

```vue
<el-button v-permission="'user:create'">
  创建用户
</el-button>
```

## 📝 注意事项

1. **权限变更生效**
   - 修改角色权限后，用户需重新登录
   - 或等待 2 小时缓存过期

2. **系统管理员**
   - 拥有所有权限
   - 不受权限检查限制

3. **审计日志**
   - 所有权限配置操作都会记录
   - 日志保留期限：永久

4. **性能优化**
   - 建议启用 Redis 缓存
   - 大量用户时考虑分页查询
