## V4.3 权限设计

### 核心表结构

| 表名 | 用途 | 字段 |
|------|------|------|
| `resource` | 资源表 | id, name, code, type, parent_id, permission_key |
| `role_permissions` | 角色 - 资源关联 | role_id, permission_id (指向 resource.id) |
| `user_roles` | 用户 - 角色关联 | user_id, role_id |

### 权限查询逻辑

```
用户权限 = 
  用户角色 (user_roles) 
  → 角色权限 (role_permissions) 
  → 资源 (resource.permission_key)
```
