## 🔐 权限系统增强

### 新增权限
| 权限代码 | 权限名称 | 描述 |
|---------|---------|------|
| `permission:role:manage` | 角色权限管理 | 管理角色权限配置 |
| `permission:user:manage` | 用户权限管理 | 管理用户权限配置 |
| `permission:edit` | 编辑权限描述 | 编辑权限描述信息 |
| `audit_log` | 审计日志 | 查看审计日志 |

### 权限保护规则
- **系统管理员角色** (`ROLE_SYSTEM_ADMIN`)：权限不可编辑，必须全部授权
- **系统管理员用户** (`admin`)：权限不可编辑
- **其他用户/角色**：可由具备权限管理权限的用户按需调整

### 后端 API 更新
- `PermissionController.java`：增加 `operationType`、`operatorName` 筛选参数
- 新增 `PUT /api/permission/{id}` 更新权限描述 API
