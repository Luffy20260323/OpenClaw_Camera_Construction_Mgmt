## 📋 根本原因

**测试用例设计错误**: AUTH-002 测试用例使用了错误的 HTTP 请求方法（POST），而接口实际支持的方法是 GET。

### 错误来源
测试指南文档中可能未明确指定请求方法，或测试脚本编写时默认使用了 POST 方法。
