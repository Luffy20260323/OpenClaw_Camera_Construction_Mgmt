## 1. 系统概述

### 1.1 系统简介

视频监控点位施工项目管理系统是一个聚焦**项目交付过程管理**和**最终结算管理**的多角色协作平台。系统支持甲方（建设方）、乙方（施工方）、监理（监督方）三方协作，实现项目从开工到施工、验收、结算、运维的全生命周期管理。

### 1.2 系统定位

- **核心功能**: 项目交付过程管理 + 最终结算管理
- **不涉及**: 投标管理、中标过程、合同谈判
- **用户角色**: 甲方（建设方）、乙方（施工方）、监理（监督方）、系统管理员
- **管理对象**: 项目 → 标段 → 点位 → 零部件
- **生命周期**: 开工 → 施工 → 审核 → 验收 → 结算 → 运维

### 1.3 技术特点

- **PC 端**: 兼容 Edge、Chrome、360、Firefox 等主流浏览器
- **移动端**: 兼容 iOS、Android、HarmonyOS
- **安全认证**: JWT Token + 双重验证码（页面验证码/手机短信验证码）
- **权限控制**: 基于角色的访问控制（RBAC）+ 作业区数据隔离

## 3. 用户注册与登录

### 3.1 用户注册

#### 3.1.1 注册流程

1. 访问系统登录页面，点击"**注册**"按钮
2. 填写注册信息：
   - **用户名**: 唯一标识，建议使用英文或拼音
   - **密码**: 符合复杂度要求（至少 8 位，包含大小写字母、数字、特殊字符）
   - **邮箱**: 用于接收通知和找回密码
   - **手机号码**: 用于接收短信验证码
   - **所属公司**: 从下拉列表选择
   - **期望角色**: 可多选（根据公司业务类型显示对应角色）
3. **甲方作业区角色**需选择归属作业区
4. 提交注册申请，系统提示"**注册申请已提交，请等待管理员审批**"

#### 3.1.2 角色类型说明

| 公司类型 | 可选角色 |
|----------|----------|
| **甲方** | 甲方管理员、甲方项目经理、甲方作业区项目主管、甲方作业区项目经理、甲方作业区项目经办人 |
| **乙方** | 乙方管理员、乙方项目经理、乙方施工队长、乙方施工人员、乙方材料管理员、乙方安全专员 |
| **监理方** | 监理方管理员、监理方项目经理、监理方监理工程师、监理方监理员 |

#### 3.1.3 审批流程

- 系统自动将注册申请推送给**公司管理员**和**系统管理员**
- 管理员可审批通过、否决或调整申请的角色
- 审批结果通过**邮箱**和**短信**通知用户

### 3.2 用户登录

#### 3.2.1 登录步骤

1. 访问系统登录页面
2. 输入**用户名**和**密码**
3. 选择验证码类型：
   - **页面验证码**: 输入页面上显示的图形验证码
   - **手机短信验证码**: 点击"获取短信验证码"，输入收到的短信验证码
4. 点击"**登录**"按钮

#### 3.2.2 登录成功

- 系统自动跳转至**任务跟踪仪表盘**（首页）
- 根据用户角色和权限显示对应的功能模块

#### 3.2.3 登录失败处理

| 错误提示 | 原因 | 解决方案 |
|----------|------|----------|
| 用户名或密码错误 | 输入错误 | 检查用户名和密码，注意大小写 |
| 验证码错误 | 验证码过期或输入错误 | 重新获取验证码 |
| 账号未审批 | 注册后管理员尚未审批 | 联系公司管理员或系统管理员 |
| 账号已禁用 | 账号被管理员禁用 | 联系系统管理员 |
| 账号已锁定 | 多次登录失败 | 等待 30 分钟后重试或联系管理员 |

## 4. 用户管理

### 4.1 用户列表

#### 4.1.1 查看用户列表

1. 点击顶部导航栏"**用户管理**" → "**用户列表**"
2. 系统分页展示所有用户信息
3. 支持以下筛选条件：
   - **关键词搜索**: 用户名、姓名、邮箱、手机号
   - **公司类型**: 甲方/乙方/监理方
   - **审批状态**: 待审批/已通过/已否决
   - **用户状态**: 正常/禁用/锁定

#### 4.1.2 用户信息展示

| 字段 | 说明 |
|------|------|
| 用户名 | 唯一标识 |
| 姓名 | 真实姓名 |
| 公司 | 所属公司 |
| 角色 | 分配的角色（可多个） |
| 作业区 | 归属作业区（仅甲方作业区角色） |
| 审批状态 | 待审批/已通过/已否决 |
| 状态 | 正常/禁用/锁定 |
| 创建时间 | 账号创建时间 |

### 4.2 创建用户

#### 4.2.1 手动创建（管理员）

1. 点击"**创建用户**"按钮
2. 填写用户信息：
   - 用户名、密码、姓名、邮箱、手机号
   - 所属公司（系统管理员可创建任意公司用户）
   - 分配角色（可多选）
   - 作业区（仅甲方作业区角色需要）
3. **系统管理员创建的用户自动审批通过**
4. 公司管理员创建的用户需系统管理员审批

#### 4.2.2 批量导入

1. 点击"**批量导入**"按钮
2. 下载 Excel 模板文件
3. 按模板格式填写用户信息
4. 上传 Excel 文件
5. 系统校验数据并导入
6. 导入结果提示成功/失败记录数

### 4.3 编辑用户

1. 在用户列表中找到目标用户
2. 点击"**编辑**"按钮
3. 修改用户信息：
   - 姓名、邮箱、手机号
   - 角色（可增删）
   - 作业区（仅甲方作业区角色）
   - 用户状态（正常/禁用）
4. 点击"**保存**"

### 4.4 删除用户

1. 在用户列表中找到目标用户
2. 点击"**删除**"按钮
3. 确认删除操作
4. **注意**: 删除后不可恢复

### 4.5 重置密码

1. 在用户列表中找到目标用户
2. 点击"**重置密码**"按钮
3. 输入新密码
4. 系统自动发送密码重置通知到用户邮箱

### 4.6 用户详情

1. 在用户列表中点击用户名
2. 查看用户完整信息：
   - 基本信息
   - 角色列表
   - 操作日志
   - 审批历史

## 11. 常见问题

**Q1: 注册后多久能收到审批通知？**

A: 系统会立即推送通知给公司管理员和系统管理员。通常 1-2 个工作日内完成审批。

**Q2: 忘记密码怎么办？**

**Q3: 为什么我登录时看不到某些功能模块？**

A: 系统根据用户角色显示功能模块。如果您需要访问某些功能，请联系管理员分配对应角色。

### 10.2 权限相关

**Q4: 为什么我不能删除某个用户/公司/角色？**

A: 系统保护的公司、角色不可删除。普通用户只能管理自己权限范围内的数据。

**Q5: 甲方作业区角色和普通甲方角色有什么区别？**

A: 甲方作业区角色只能查看和管理所属作业区的项目和点位，普通甲方角色可以查看和管理所有项目。

### 10.3 浏览器兼容

**Q6: 为什么 Chrome 浏览器显示空白页面？**

A: 可能是浏览器缓存问题。尝试：
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 使用无痕模式访问
3. 尝试 Firefox 浏览器

**Q7: 移动端可以访问吗？**

A: 系统支持移动端访问，兼容 iOS、Android、HarmonyOS。推荐使用最新版浏览器。
