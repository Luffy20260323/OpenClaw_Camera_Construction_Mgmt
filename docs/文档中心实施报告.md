## 🔐 安全控制

### 访问控制

1. **登录验证**
   - 所有文档 API 都需要 JWT Token
   - 未登录用户无法访问

2. **权限验证**
   - 使用 `@PreAuthorize("hasAuthority('system:docs:view')")`
   - 只有授权用户才能访问

3. **路径安全**
   - 防止路径遍历攻击（`..` 检查）
   - 限制只能访问 docs 目录

4. **文件类型限制**
   - 只允许访问 HTML、Markdown、PDF
   - 排除 node_modules、target 等目录


