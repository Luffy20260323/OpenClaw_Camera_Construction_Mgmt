# 前端权限测试用例

> **项目：** OpenClaw_Camera_Construction_Mgmt  
> **测试范围：** 前端权限控制  
> **创建日期：** 2026-04-01  
> **测试负责人：** Camera1001 Team

---

## 测试目标

1. 验证前端页面权限控制是否正确生效
2. 验证未授权用户访问受限页面时的行为（应跳转/提示）
3. 验证菜单项权限过滤是否正确
4. 验证按钮/操作权限控制
5. 验证 API 权限拦截

---

## 测试环境

| 配置项 | 值 |
|--------|-----|
| 前端框架 | Vue 3 + Vite |
| UI 组件库 | Element Plus |
| 状态管理 | Pinia |
| 路由管理 | Vue Router 5 |
| 权限指令 | v-permission, v-menu-permission |

---

## 测试用户角色

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 系统管理员 (system_admin) | 拥有所有权限 | *:*:* |
| 普通用户 | 仅有基础权限 | home, profile |
| 项目经理 | 有项目管理权限 | project_management, workarea_management |
| 审计员 | 仅有审计日志查看权限 | audit_log, permission_audit_log |
