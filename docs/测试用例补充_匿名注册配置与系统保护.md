# 测试用例补充 - 匿名注册配置与系统保护

**版本号**: v5.0-Anonymous  
**日期**: 2026-03-21  
**编制**: OpenClaw AI Assistant  
**补充说明**: 测试公司匿名注册配置的前端显示逻辑和系统保护公司的配置限制  

---

## 🎯 测试目标

验证**公司匿名注册配置**的两个关键场景：

1. **登录页面注册表单**：只有允许匿名注册的公司才出现在公司列表中
2. **系统保护公司**：系统管理员所在公司禁止修改匿名注册配置（必须一直禁止）

## 🔐 安全测试要点

### 前端验证
- 登录页面注册表单的公司列表动态过滤
- 只显示 allowAnonymousRegister=true 的公司
- 系统保护公司的配置字段禁用

### 后端验证
- 注册 API 验证公司是否允许匿名注册
- 修改公司 API 验证系统保护
- 返回适当的错误码（400 或 403）

### 数据库验证
- 系统保护公司的 allowAnonymousRegister 始终为 false
- 即使数据库被直接修改，业务逻辑也应阻止

### 多层防护
- 前端过滤 + 后端验证 + 数据库约束
- 系统保护公司有特殊标记（is_system_protected）
- 业务逻辑检查系统保护标记
