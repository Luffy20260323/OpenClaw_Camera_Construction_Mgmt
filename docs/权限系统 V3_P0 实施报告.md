## 🔐 核心功能实现

### 1. 权限分组管理

```sql
-- 按模块分组管理权限
SELECT group_code, group_name, COUNT(*) as permission_count
FROM permissions
GROUP BY group_code, group_name
ORDER BY sort_order;
```

### 2. 权限类型区分

```sql
-- VIEW 类型（查看类）
SELECT * FROM permissions WHERE permission_type = 'VIEW';

-- ACTION 类型（操作类）
SELECT * FROM permissions WHERE permission_type = 'ACTION';
```

### 3. 基本权限集合

```sql
-- 所有角色必备的权限
SELECT permission_code, permission_name 
FROM permissions 
WHERE is_base = TRUE;

-- 结果：
-- auth:login, auth:logout
-- user:profile:view, user:profile:edit
-- user:password:change
```

### 4. 角色层级管理

```sql
-- 查看角色层级
SELECT role_code, role_name, role_level, managed_by_role_id
FROM roles
ORDER BY role_level;

-- 系统管理员：Level 0（最高）
-- 其他角色：待配置
```

## 📝 技术细节

### 权限编码规范

```
格式：{模块}:{实体}:{操作类型}

示例：
auth:login:action          - 登录系统
user:profile:view          - 查看个人资料
user:create:action         - 创建用户
```

### 权限类型定义

| 类型 | 说明 | 示例 |
|------|------|------|
| VIEW | 查看类权限（只读） | user:list:view |
| ACTION | 操作类权限（增删改） | user:create:action |

### 角色层级定义
