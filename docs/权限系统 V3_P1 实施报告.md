## 🔐 核心功能实现

### 1. 页面元素权限控制

```sql
-- 查询某个页面的所有元素及所需权限
SELECT 
    p.page_name,
    e.element_name,
    e.element_type,
    perm.permission_code,
    perm.permission_name
FROM pages p
JOIN page_elements e ON p.id = e.page_id
JOIN element_permissions ep ON e.id = ep.element_id
JOIN permissions perm ON ep.permission_id = perm.id
WHERE p.page_code = 'user_list';

-- 结果：
-- 用户列表 | 新增按钮 | button | user:create | 创建用户
-- 用户列表 | 编辑按钮 | button | user:edit | 编辑用户
-- 用户列表 | 删除按钮 | button | user:delete | 删除用户
```

### 2. 前端权限控制示例

```vue
<!-- 按钮级别的权限控制 -->
<template>
  <el-button v-permission="'user:create'">新增</el-button>
  <el-button v-permission="'user:edit'">编辑</el-button>
  <el-button v-permission="'user:delete'" type="danger">删除</el-button>
</template>

<!-- 权限指令 -->
<script>
app.directive('permission', {
  mounted(el, binding) {
    const permission = binding.value
    const permissions = useUserStore().permissions
    if (!permissions.includes(permission)) {
      el.parentNode?.removeChild(el)
    }
  }
})
</script>
```

### 3. 权限配置权管理

```sql
-- 查询用户的权限配置权
SELECT 
    u.username,
    pcr.can_config_base,
    pcr.can_config_default,
    pcr.can_reset_default,
    pcr.can_adjust_user,
    pcr.can_manage_roles
FROM users u
JOIN permission_config_rights pcr ON u.id = pcr.user_id
WHERE u.username = 'admin';

-- 结果：
-- admin | true | true | true | true | true
```

### 4. 角色缺省权限

```sql
-- 查询角色的缺省权限
SELECT 
    r.role_name,
    p.permission_code,
    p.permission_name,
    rdp.is_inherited,
    rdp.configured_at
FROM roles r
JOIN role_default_permissions rdp ON r.id = rdp.role_id
JOIN permissions p ON rdp.permission_id = p.id
WHERE r.role_code = 'ROLE_SYSTEM_ADMIN'
LIMIT 10;
```

### 5. 权限快照与回滚

```sql
-- 创建权限快照
INSERT INTO permission_snapshots (
    snapshot_name, snapshot_type, target_type, target_id,
    snapshot_data, created_by
) VALUES (
    '角色权限变更前快照',
    'ROLE',
    'ROLE',
    1,
    json_build_object(
        'permissions', 
        (SELECT json_agg(permission_id) FROM role_permissions WHERE role_id = 1)
    ),
    1
);

-- 回滚到快照
UPDATE permission_snapshots 
SET rolled_back = TRUE, rolled_back_at = NOW(), rolled_back_by = 1
WHERE id = 1;
```
